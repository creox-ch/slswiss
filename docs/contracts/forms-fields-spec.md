## Общие правила (все формы)

- Обязательные поля помечены `*` в UI (как в регистрации), submit заблокирован/ошибка под полем.
- Лимиты длины: заголовок ≤200, описание ≤5000, обрезка/ошибка (уже частично есть).
- Кантон — **единый источник** (см. `cantons-fix`), одинаковый в регистрации и во всех формах.
- Спецсимволы/эмодзи — сохраняются (UTF-8, экранирование при выводе `_pubEsc`).
